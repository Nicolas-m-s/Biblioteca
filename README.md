Sistema de Gestión de Biblioteca

Aplicación desarrollada en Java para la administración de libros dentro de una biblioteca mediante consola.

El sistema permite registrar libros, buscarlos, prestarlos y consultar su disponibilidad utilizando programación orientada a objetos.


Características

- Agregar nuevos libros
- Buscar libros por título
- Prestar libros
- Consultar disponibilidad
- Listar todos los libros registrados
- Interfaz interactiva por consola

Tecnologías utilizadas

- Java
- Programación Orientada a Objetos (POO)
- ArrayList
- Scanner
- Estructuras de control



Estructura del proyecto

```text
Main.java
 ├── Libro
 ├── Biblioteca
 └── Main
```

Clases principales

`Libro`
Representa cada libro del sistema.

Atributos:
- Título
- Autor
- Estado de préstamo


`Biblioteca`
Gestiona la colección de libros.

Funciones:
- Agregar libros
- Buscar libros
- Prestar libros
- Listar catálogo


`Main`
Controla la interacción con el usuario mediante menú por consola.


Cómo ejecutar

1. Clonar el repositorio

```bash
git clone URL_DEL_REPOSITORIO
```

2. Compilar

```bash
javac Main.java
```

3. Ejecutar

```bash
java Main
```
